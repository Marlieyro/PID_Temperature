In this mini-project I practiced configuring peripherals without CubeMX, using HAL. 
I already have experience in configuring and using peripherals using Low Level Lib, and the experience with HAL was new. 

This is a simple project in which I set the temperature with a controller and compare it with a BME sensor that measures the fan temperature. 
This project is purely practical, the only new thing I learned is the PID formula. 

I don't have a cooler, so I used the HIL mechanism.

Also figured out the library for the BME sensor
